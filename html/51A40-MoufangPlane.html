Moufang plane


A projective plane is called a Moufang plane if each of its lines is a translation line (it is a translation plane with respect to any line in the plane). In other words, a Moufang plane is a projective plane such that the minor Desarguesian property holds everywhere.

It can be shown that a projective plane is Moufang iff it can be coordinatized by an alternative division ring, which is a non-associative division ring satisfying the left and right alternative laws:
