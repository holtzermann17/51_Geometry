betweenness relation


1 Definition

Let A be a set. A ternary relation B on A is said to be a betweenness relation if it has the following properties:

  1. O1

    if (a,b,c)∈B, then (c,b,a)∈B; in other words, the set

    B⁢(b)={(a,c)∣(a,b,c)∈B}

    is a symmetric relation (http://planetmath.org/Symmetric) for each b; thus, from now on, we may say, without any ambiguity, that b is between a and c if (a,b,c)∈B;

  2. O2

    if (a,b,a)∈B, then a=b;

  3. O3

    for each a,b∈A, there is a c∈A such that (a,b,c)∈B;

  4. O4

    for each a,b∈A, there is a c∈A such that (a,c,b)∈B;

  5. O5

    if (a,b,c)∈B and (b,a,c)∈B, then a=b;

  6. O6

    if (a,b,c)∈B and (b,c,d)∈B, then (a,b,d)∈B;

  7. O7

    if (a,b,d)∈B and (b,c,d)∈B, then (a,b,c)∈B.
