piecewise smooth


A curve α:[a,b]→ℝn is said to be piecewise smooth if each component α1,…,αn of α has a bounded derivative αi′ (i=1,…,n) which is continuous everywhere in [a,b] except (possibly) at a finite number of points at which left- and right-sided derivatives exist.

  • •

    Every piecewise smooth curve is rectifiable.

  • •

    Every rectifiable curve can be approximated by piecewise smooth curves.
