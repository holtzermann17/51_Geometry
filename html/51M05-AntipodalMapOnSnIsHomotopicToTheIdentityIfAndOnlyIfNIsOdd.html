antipodal map on Sn is homotopic to the identity if and only if n is odd


Lemma.

If X:Sn→Sn is a unit vector field, then there is a homotopy between the antipodal map on Sn and the identity map.

Proof.

Regard Sn as a subspace of Rn+1 and define H:Sn×[0,1]→Rn+1 by H⁢(v,t)=(cos⁡π⁢t)⁢v+(sin⁡π⁢t)⁢X⁢(v). Since X is a unit vector field, X⁢(v)⟂v for any v∈Sn. Hence ∥H⁢(v,t)∥=1, so H is into Sn. Finally observe that H⁢(v,0)=v and H⁢(v,1)=-v. Thus H is a homotopy between the antipodal map and the identity map. ∎

Proposition.

The antipodal map A:Sn→Sn is homotopic to the identity if and only if n is odd.

Proof.

If n is even, then the antipodal map A is the composition of an odd of reflections. It therefore has degree -1. Since the degree of the identity map is +1, the two maps are not homotopic.

Now suppose n is odd, say n=2⁢k-1. Regard Sn has a subspace of ℝ2⁢k. So each point of Sn has coordinates (x1,…,x2⁢k) with ∑ixi2=1. Define a map X:ℝ2⁢k→ℝ2⁢k by X⁢(x1,x2,…,x2⁢k-1,x2⁢k)=(-x2,x1,…,-x2⁢k,x2⁢k-1), pairwise swapping coordinates and negating the even coordinates. By construction, for any v∈Sn, we have that ∥X⁢(v)∥=1 and X⁢(v)⟂v. Hence X is a unit vector field. Applying the lemma, we conclude that the antipodal map is homotopic to the identity. ∎
