homology of the sphere


Every loop on the sphere S2 is contractible to a point, so its fundamental group, π1⁢(S2), is trivial.

Let Hn⁢(S2,ℤ) denote the n-th homology group of S2. We can compute all of these groups using the basic results from algebraic topology:

  • •

    S2 is a compact orientable smooth manifold, so H2⁢(S2,ℤ)=ℤ;

  • •

    S2 is connected, so H0⁢(S2,ℤ)=ℤ;

  • •

    H1⁢(S2,ℤ) is the abelianization of π1⁢(S2), so it is also trivial;

  • •

    S2 is two-dimensional, so for k>2, we have Hk⁢(S2,ℤ)=0
