Veblen-Wedderburn system


A Veblen-Wedderburn system is an algebraic system over a set R with two binary operations + (called addition) and ⋅ (called multiplication) on R such that

  1. 1.

    there is a 0∈R, and that R is an abelian group under +, with 0 the additive identity

  2. 2.

    R-{0}, together with ⋅, is a loop (we denote 1 as its identity element)

  3. 3.

    ⋅ is right distributive over +; that is, (a+b)⋅c=a⋅c+b⋅c

  4. 4.

    if a≠b, then the equation x⋅a=x⋅b+c has a unique solution in x

A Veblen-Wedderburn system is also called a quasifield.

Usually, we write a⁢b instead of a⋅b.

For any a,b,c∈R, by defining a ternary operation * on R, given by

a*b*c:=a⁢b+c,

it is not hard to see that (R,*,0,1) is a ternary ring. In fact, it is a linear ternary ring because a⁢b=a*b*0 and a+c=a*1*c.

For example, any field, or more generally, any division ring, associative or not, is Veblen-Wedderburn. An example of a Veblen-Wedderburn system that is not a division ring is the Hall quasifield.

A well-known fact about Veblen-Wedderburn systems is that, the projective plane of a Veblen-Wedderburn system is a translation plane, and, conversely, every translation plane can be coordinatized by a Veblen-Wedderburn system. This is the reason why a translation plane is also called a Veblen-Wedderburn plane.

Remark. Let R be a Veblen-Wedderburn system. If the multiplication ⋅, in addition to be right distributive over +, is also left distributive over +, then R is a semifield. If ⋅, on the other hand, is associative, then R is an abelian nearfield (a nearfield such that + is commutative).
