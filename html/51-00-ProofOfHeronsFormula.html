proof of Heron’s formula


Let α be the angle between the sides b and c, then we get from the cosines law:

cos⁡α=b2+c2-a22⁢b⁢c.

Using the equation

sin⁡α=1-cos2⁡α

we get:

sin⁡α=-a4-b4-c4+2⁢b2⁢c2+2⁢a2⁢b2+2⁢a2⁢c22⁢b⁢c.

Now we know:

Δ=12⁢b⁢c⁢sin⁡α.

So we get:

Δ = 14⁢-a4-b4-c4+2⁢b2⁢c2+2⁢a2⁢b2+2⁢a2⁢c2
= 14⁢(a+b+c)⁢(b+c-a)⁢(a+c-b)⁢(a+b-c)
= s⁢(s-a)⁢(s-b)⁢(s-c).

This is Heron’s formula. □
